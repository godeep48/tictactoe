# tictactoe
Tic-tac-toe is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game. The code has been created in such a way that no Player can defeat the computer.
Remember this is a backend of TicTacToa, which means it lacks Graphical user interface, but it display basic TUI to make it understandable.

# How to play
User need to enter the numbers of field to make the players turn whenever it is prompted. The field number is in the pattern of keypad dialer.
